# Calculator

A simple command-line calculator application built with Python that performs basic arithmetic operations including addition, subtraction, multiplication, and division.

## Features

- Addition, subtraction, multiplication, and division operations
- User-friendly command-line interface
- Input validation and error handling
- Clear output formatting

## How to Run

1. Make sure you have Python 3 installed on your system
2. Navigate to the CALCULATOR directory
3. Run the script using the command: `python main.py`
4. Follow the on-screen prompts to enter numbers and select operations

## Usage

1. Enter two numbers when prompted
2. Select an operation from the menu:
   - `+` for addition
   - `-` for subtraction
   - `*` for multiplication
   - `/` for division
3. The result will be displayed on the screen

## Error Handling

- The application handles division by zero errors
- Invalid operations are caught and reported
- Input validation ensures numeric values are processed correctly

## File Structure

- `main.py` - The main application file containing the calculator logic

## Learning Concepts Demonstrated

- User input handling with `input()`
- Conditional logic using `match` statements (Python 3.10+)
- Exception handling with `try-except` blocks
- Basic arithmetic operations
- String formatting with f-strings
